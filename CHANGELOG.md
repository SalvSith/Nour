## [1.0.0] - 2026-04-12

### Added

- Voice-driven WebGL orb (React, Vite, TypeScript) with real-time emotion-driven visuals.
- Emotion pipeline: WebM capture, Groq Whisper + Llama via Node dev server or Supabase Edge Function.
- Trauma, love/euphoria, tricks, lava-lamp mini-game, `/curious` route, localStorage persistence.
- MIT license, project documentation, and contributor / release tooling (this changelog, Dependabot, contributing guide).
